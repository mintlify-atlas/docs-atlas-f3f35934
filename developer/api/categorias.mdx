---
title: "GET /api/categorias"
description: "Retrieve the full product category catalog from the SharePoint CMS."
---

Returns all published categories and the catalog timestamp. Data is sourced from `cms/catalog.json`, which is generated by the SharePoint sync script at build time and refreshed via ISR at runtime.

**Method:** `GET`  
**Path:** `/api/categorias`  
**Caching:** SWR 300 s (5 minutes)

<Note>
  This route is covered by the `"/api/categorias/**": { swr: 300 }` rule in `nuxt.config.ts`. Responses are served stale while Nitro revalidates in the background every 5 minutes.
</Note>

## Response

### 200 OK

<ResponseField name="categories" type="object[]" required>
  Array of published category objects.

  <Expandable title="category fields">
    <ResponseField name="id" type="string">
      SharePoint list item ID.
    </ResponseField>
    <ResponseField name="slug" type="string">
      URL segment used to build the category path (e.g. `gran-formato`).
    </ResponseField>
    <ResponseField name="path" type="string">
      Full path for the category page (e.g. `/categorias/gran-formato`).
    </ResponseField>
    <ResponseField name="parent" type="string | undefined">
      Slug of the parent category. Absent for top-level categories.
    </ResponseField>
    <ResponseField name="type" type="string">
      Either `"categoria"` (top-level) or `"subcategoria"` (nested).
    </ResponseField>
    <ResponseField name="title" type="string">
      Display title of the category.
    </ResponseField>
    <ResponseField name="nav" type="string">
      Short label used in navigation menus.
    </ResponseField>
    <ResponseField name="order" type="number">
      Sort order relative to sibling categories.
    </ResponseField>
    <ResponseField name="hidden" type="boolean">
      Whether the category is hidden from navigation.
    </ResponseField>
    <ResponseField name="featured" type="boolean">
      Whether the category appears in featured/promoted slots.
    </ResponseField>
    <ResponseField name="isPublished" type="boolean">
      Publication status. Only published categories are included.
    </ResponseField>
    <ResponseField name="description" type="string | undefined">
      Short text description.
    </ResponseField>
    <ResponseField name="image" type="object">
      <Expandable title="image fields">
        <ResponseField name="image.src" type="string | undefined">
          Image URL (typically from Azure Blob CDN).
        </ResponseField>
        <ResponseField name="image.width" type="number | undefined">
          Image width in pixels.
        </ResponseField>
        <ResponseField name="image.height" type="number | undefined">
          Image height in pixels.
        </ResponseField>
        <ResponseField name="image.alt" type="string">
          Alt text for the image.
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="seo" type="object">
      <Expandable title="seo fields">
        <ResponseField name="seo.metaTitle" type="string">
          `<title>` tag value.
        </ResponseField>
        <ResponseField name="seo.metaDescription" type="string">
          Meta description.
        </ResponseField>
        <ResponseField name="seo.canonical" type="string">
          Canonical URL.
        </ResponseField>
        <ResponseField name="seo.schema" type="object">
          JSON-LD structured data (`CollectionPage` schema).
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="tabs" type="object[]">
      Content tabs for the category page. Generated from `TabsJson` or parsed from `BodyMd` H2 sections.
    </ResponseField>
    <ResponseField name="faqs" type="object[]">
      FAQ items for the category page.
    </ResponseField>
    <ResponseField name="updatedAt" type="string | null">
      ISO 8601 timestamp of the last SharePoint modification.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="updatedAt" type="string | null" required>
  ISO 8601 timestamp of when the catalog was last synced from SharePoint. `null` if the catalog has never been synced.
</ResponseField>

## Navigation endpoint

For menu rendering, use the companion navigation endpoint:

**`GET /api/nav/categorias`**

This endpoint returns a tree structure with `children` arrays and an `indexBySlug` map, optimised for building nested navigation menus. It accepts a `productLimit` query parameter (1–12, default 8) to include a limited product list per category node.

```typescript
// composables/useCategoriasNav.ts
export function useCategoriasNav(opts: { productLimit?: number } = {}) {
  return useFetch<{ tree: CategoriaNode[]; indexBySlug: Record<string, CategoriaNode>; menuItems: CategoriaNode[] }>(
    "/api/nav/categorias",
    { query: { productLimit: opts.productLimit ?? 8 } }
  )
}
```
